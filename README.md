# 💃 Le Dance Bot

Telegram-бот для записи на курс танцев.

## Переменные окружения
- `BOT_TOKEN` — токен от BotFather

## Деплой на Railway
1. Загрузи папку на GitHub
2. Подключи репозиторий в Railway
3. Добавь переменную BOT_TOKEN
4. Railway сам запустит бота

## Команды для Полины
- `/addold @username1\n@username2` — добавить учеников старого потока со скидкой 13%

## Ссылки для групп
- Открытая группа: `https://t.me/ИМЯ_БОТА?start=new`
- Закрытая группа (старый поток): `https://t.me/ИМЯ_БОТА?start=vip`
